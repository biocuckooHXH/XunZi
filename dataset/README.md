# XunZi Dataset

This folder contains datasets related to the XunZi project. Below is a description of each file.

## Files Description

1. **README.md**
   - This file provides an overview of the contents of the `dataset` folder.

2. **supplementary_table1_GeneRIF_Function_Description1.zip**
   - A dataset related to gene function descriptions, part of the supplementary tables.

3. **supplementary_table1_GeneRIF_Function_Description2.zip**
   - Another dataset containing additional gene function descriptions.

4. **supplementary_table1_GeneRIF_Function_Description3.zip**
   - A third dataset containing more gene function descriptions for supplementary table 1.

5. **supplementary_table1_GeneRIF_Function_Description4.zip**
   - A fourth dataset with gene function descriptions, completing the supplementary table.

## Notes

- Ensure you have the necessary software to decompress `.zip` files.
