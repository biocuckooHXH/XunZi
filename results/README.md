# XunZi Results

This folder contains the results for various datasets and folds used in the XunZi project. The files represent different stages of data processing and model outputs, including merged datasets and cross-validation folds. Below is a description of each file in this directory.

## Files Description

1. **CTD_fold_5_for_gpt_merged.zip**
   - This file contains the merged dataset for the fifth fold of the CTD (Comparative Toxicogenomics Database) cross-validation.

2. **Disgenet_data_fold_5_CTD_only.zip**
   - This dataset contains the results of using DisGeNET data for training and CTD data for testing.

3. **Disgenet_fold_5_for_gpt_merged1.zip**
   - This file contains part of the results from the fifth fold of DisGeNET's 5-fold cross-validation, split due to large file size. 

4. **Disgenet_fold_5_for_gpt_merged2.zip**
   - This file contains the second part of the results from the fifth fold of DisGeNET's 5-fold cross-validation, also split due to large file size

5. **README.md**
   - This file provides an overview of the contents of the `results` folder.

## Notes

- Ensure you have the necessary software to decompress `.zip` and `.7z` files.
- For further details on the data preparation and model usage, please refer to the main project documentation.
